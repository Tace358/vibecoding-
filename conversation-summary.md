# 对话总结：电商智能图文生成工具开发

## 1. 主要请求与意图

用户希望创建一个帮助电商运营人员自动生成商品主图和文案的工具。核心需求包括：

- **页面布局**：左侧面板输入商品信息，右侧面板显示生成结果
- **输入功能**：支持上传图片、填写文字、上传历史爆款截图或参考链接
- **批量处理**：支持Excel批量导入或在线录入/上传
- **素材管理**：支持保存商品素材到素材库方便复用
- **输出内容**：
  - 每个商品一张"看得过去、包含基础卖点"的主图草稿
  - 一条"结构合理、含核心关键词"的标题 + 1-2句卖点文案
- **工作流程**：从白手起稿变为系统生成草稿后进行筛选和微调
- **四个主要页面**：
  1. 首页 - 展示项目概览与整体进度
  2. 商品任务列表页 - 核心功能页面
  3. 模板库页 - 查看与管理收藏的图文模板
  4. 设置页 - 配置基础参数和风格偏好

## 2. 关键技术概念

- **前端框架**：React 18+ 配合 TypeScript
- **样式方案**：Tailwind CSS 实现响应式设计
- **构建工具**：Vite
- **图标库**：lucide-react
- **状态管理**：React Hooks (useState, useRef, useCallback)
- **文件处理**：拖拽上传、Excel导入
- **设计系统**：Easy Vibe 风格
  - 圆点背景图案
  - 玻璃态效果（Glass Morphism）
  - 悬浮动画和过渡效果
  - 卡片悬浮效果
